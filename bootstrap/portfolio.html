Services Used in the Portfolio


EC2 (Elastic Compute Cloud): This is used to deploy the website, essentially providing virtual servers in the cloud.

DynamoDB: It's like a database where user reviews are stored. It's fast and efficient for handling large-scale data.

SES (Simple Email Service): It's used to receive emails whenever someone submits a contact form on the website. So, you get notified via email.

Cognito: This adds user authentication to the website. Users need to sign in or sign up to access certain media content.

Restricted API: This is an API that requires an API key to access. It adds an extra layer of security, ensuring only authorized users can access certain parts of the website.

S3 (Simple Storage Service): It's used for storing images and documents, acting as a sort of cloud storage.

Route53: It's used to assign a domain name to the website, making it easier for users to find.

CloudFront: This service helps to ensure that when someone visits the website, they are redirected from HTTP to HTTPS for secure browsing.

API Gateway: It's used to develop APIs for forms on the website, like the contact form and review form.

Lambda Function: These are small pieces of code that are triggered by certain events, like API calls. They're used to perform specific tasks in response to these events.

CloudWatch: This service is used for monitoring and logging, helping you keep track of what's happening with your website and services.
